# ApexPlanet Task 2 - Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project is part of the ApexPlanet Data Analytics Internship. The objective is to perform Exploratory Data Analysis (EDA) on an E-Commerce Sales dataset to discover business insights using Python.

## 🎯 Objectives
- Perform data exploration and cleaning
- Calculate business KPIs
- Analyze customer demographics
- Analyze product and category performance
- Analyze city-wise sales
- Perform time series analysis
- Analyze revenue distribution
- Perform correlation analysis
- Generate business insights

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab
- Git & GitHub
